# 🎤 Pitch – Finova (Assistente Financeiro com IA)

## 1. 🧩 O Problema (30s)

Muitas pessoas têm dificuldade em entender para onde seu dinheiro está indo.

Mesmo com acesso a extratos e aplicativos bancários, falta clareza sobre:
- padrões de consumo
- excesso de gastos
- organização financeira

Além disso, a maioria das pessoas não tem acesso fácil a orientação financeira personalizada.

---

## 2. 💡 A Solução (1 min)

O Finova é um assistente financeiro inteligente que utiliza IA para analisar dados reais do usuário e gerar insights personalizados.

A solução funciona assim:

- O usuário interage via chatbot (Streamlit)
- O sistema analisa dados como:
  - transações financeiras (CSV)
  - perfil do investidor (JSON)
- Um modelo de linguagem (Llama 3.2 via Ollama) processa essas informações localmente
- O agente responde com base nos dados, evitando alucinações

Diferente de soluções genéricas, o Finova:
- responde com base em dados reais do usuário
- respeita o perfil financeiro
- roda 100% local, garantindo privacidade

---

## 3. 🖥️ Demonstração (1 min)

Na demonstração, será mostrado:

1. Interface do chatbot no Streamlit
2. Usuário fazendo uma pergunta como:
   "Estou gastando muito com alimentação?"
3. O sistema analisando as últimas transações
4. O agente retornando uma resposta baseada nos dados

Também será demonstrado:
- uma pergunta fora do escopo (ex: clima)
- e como o agente responde de forma segura

---

## 4. 🚀 Diferencial e Impacto (30s)

O principal diferencial do Finova é:

- Uso de IA local (via Ollama) → mais privacidade e zero custo por requisição
- Controle de alucinação com prompt estruturado
- Integração com dados reais (não é só um chatbot genérico)

Impacto:

O Finova ajuda usuários a:
- entender melhor seus hábitos financeiros
- tomar decisões mais conscientes
- dar os primeiros passos rumo à educação financeira
